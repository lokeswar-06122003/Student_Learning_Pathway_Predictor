# 🎓 Student Learning Pathway Predictor

A Machine Learning based web application that predicts the most suitable learning pathway for students based on their academic performance and educational background.

---

## 📌 Project Overview

The Student Learning Pathway Predictor helps students identify the learning pathway that best matches their academic profile. The application uses a trained Machine Learning model to analyze student information and recommend an appropriate stream.

The project is developed using **Python**, **Flask**, providing an interactive web interface for real-time predictions.

---

## 🚀 Features

- Predicts suitable learning pathways
- User-friendly web interface
- Machine Learning based prediction
- Data preprocessing and cleaning
- Trained classification model
- Fast real-time predictions using Flask

## 📊 Input Features

- Gender
- No.of hours studied per week
- Maths Marks
- Science Marks
- Arts Marks

## 🎯 Prediction Output


